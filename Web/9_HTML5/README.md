## 시맨틱 태그
- header : 머리말을 나타냄
- hgroup : 제목과 부제목을 묶음
- nav : 메뉴 부분을 나타냄
- article : 개별 컨텐츠를 나타냄
- section : 제목별로 나눔
- aside : 좌우측의 사이드바를 나타냄
- footer : 제작자의 정보나 저작권의 정보를 나타냄

#
## 추가 HTML5 태그
- mark : 특정 텍스트를 강조
- time : 날짜, 시간 표현에 의미를 부여
- meter, progress : 일정 범위 안의 값이나 분표 비율 등을 나타낼 때 사용
- a download : 다운로드 가능
